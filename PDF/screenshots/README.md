# CanCan Campus Radio 截图替换说明

当前 PDF 先使用 TeX 占位图，不需要真实截图也可以编译。等你准备最终版时，把截图放到这个目录，我再把 `main.tex` 里的占位图替换为真实图片。

建议截图尺寸：

- 优先使用 16:9，例如 1920x1080、1600x900 或 1366x768。
- 浏览器缩放保持 100%。
- 尽量只保留产品界面，不要截到桌面、文件夹、隐私信息或密钥。
- 可以用 Windows 的 `Win + Shift + S` 截图，保存为 PNG。

请按下面文件名保存：

1. `01-home.png`
   - 首页或电台主界面。
   - 建议展示 AI DJ 头像、电台状态、当前播放或启动电台入口。

2. `02-radio-chat.png`
   - AI DJ 对话和推荐界面。
   - 建议展示用户输入心情或场景后，AI DJ 给出歌曲与口播。

3. `03-library-profile.png`
   - 曲库、用户音乐画像或同步结果。
   - 建议展示产品如何理解用户音乐偏好。

4. `04-mixer-feedback.png`
   - 反馈、喜欢/不喜欢、下一首、调音台或类似交互。
   - 建议展示用户如何让电台越用越懂自己。

截图完成后告诉我，我会把占位图替换成真实截图并重新编译 PDF。
