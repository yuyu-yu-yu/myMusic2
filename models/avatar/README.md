# Avatar segmentation model

The unification pipeline uses `isnetis.onnx` from:

- https://huggingface.co/skytnt/anime-seg
- License: Apache-2.0

The model file is intentionally ignored by Git because it is about 168 MB.
Run the setup command to create the isolated Python environment and download it:

```powershell
npm run avatar:unify:setup
```
